\# Java To-Do List Application



A simple To-Do List application built using Java and Swing.



\## Features

\- Add tasks

\- Delete tasks

\- Clean GUI

\- Custom font support



\## Technologies Used

\- Java

\- Swing



\## How to Run

1\. Clone the repository

2\. Open in IntelliJ IDEA

3\. Run `App.java`
